# 中文APA格式规则（《心理学报》著者-出版年制）

> 依据：Publication Manual of the American Psychological Association 第7版（2019）中文细化规定

---

## 一、总体要求

1. 正文引用与文后文献列表必须完全一致。
2. 文献列表按著者姓氏汉语拼音字母顺序排列；同姓按名排序；同姓同名按出版年排序。
3. 相同著者、相同出版年的不同文献，在年份后加 a、b、c……区分，按文题字母顺序排列。
   - 示例：Wang, M. Y. (2008a). Emotional… / Wang, M. Y. (2008b). Monitor…
4. 中文文献应给出原文的英文信息（抄录原文，不得擅自翻译），英文在前，中文放在下方方括号中。

---

## 二、正文内引用格式

### 2.1 基本格式

| 情形 | 作为句子成分 | 放在括号中 |
|------|-------------|-----------|
| 1位作者 | 张三(2008)发现… | (张三, 2008) |
| 1位英文作者 | Dell (1986)提出… | (Dell, 1986) |
| 2位作者（中文） | 张三和李四(2008) | (张三, 李四, 2008) |
| 2位作者（英文，括号内） | — | (Wolchik & West, 2007) |
| 2位作者（英文，句子成分） | Wang和Sun (2009) | — |
| 3位及以上作者 | 张三等人(2008) | (张三 等, 2008) |
| 3位及以上英文作者 | Wang et al. (2009) | (Wolchik et al., 2007) |

### 2.2 特殊规则

- **重复引用**：同一段落中同一文献连续引用，第二次及以后可省略出版年。
- **作者同姓**：如有两篇文献作者同姓不同名，需加名的首字母区分，如 K. D. Wang (2007) 和 P. G. Wang (2008)。
- **多篇文献同时引用**：同一括号内，先按字母序排列，用分号隔开；如 (Bai, 2004; Chen, 2006; Deng & Fang, 2005)。
- **et al. 的限制**：若只剩最后一位作者无法用 et al.（因为 et al. 为复数），则直接写出该作者姓名。
- **括号内中文多作者**：用逗号隔开，**不**加"&"，如 (张三, 李四, 王五, 2019)。
- **括号内英文多作者**：最后两位之间用"&"，倒数第二位后仍有逗号，如 (Qian, Zhao, Zhou, & Sun, 2019)。

---

## 三、文献列表著录规则

### 3.1 著者格式

- 姓全拼，名只写首字母；姓后逗号，名缩写字母后有缩写点；**姓前名后**。
  - 示例：Zhang, X. L.
- 2–7位著者：全部列出，最后两位之间用"&"（英文）；中文著者之间用逗号，最后两位之间也用逗号（不加"&"）。
- 超过7位著者：列出前6位，省略号，再列出最后1位。
- 有连字符的名字保留连字符：Hau, K.-T.
- 团体著者全称，大单位在前小单位在后：Beijing Normal University, School of Psychology.
- 无著者时，文题/书名前移至著者位置。
- 编辑图书：著者姓名后括号内加 Ed. 或 Eds.；中文加"编"或"主编"。

### 3.2 出版日期格式

| 类型 | 格式 |
|------|------|
| 学术期刊、图书、音像 | 只写年份，如 (2008). |
| 会议论文集、非学术杂志、快报 | 年和月，如 (2008, 2月). |
| 日报、周报 | 年月日，如 (2008-02-08). |
| 已接受未印刷 | in press，如 (in press). |
| 无日期 | (n.d.). |

### 3.3 文题格式

- 期刊论文文题：**正体**（非斜体），句式大写（仅首词及专有名词大写）。
- 书名：**斜体**，句式大写。
- 特殊文献类型在题名后方括号内标注，如 [宣传册]、[录像带]、[软件]、[数据]。

### 3.4 期刊信息格式

```
著者. (出版年). 文题. 刊名（斜体）, 卷号（斜体）(期号), 页码范围.
```

- 刊名与卷号用斜体，期号不用斜体。
- 页码范围符号用破折号"−"，**不是**连字符"-"。
- 刊名、卷号、页码之间用逗号隔开。
- 最后一位著者与出版年括号之间留一空格。

**示例：**
```
Zhuang, J., & Zhou, X. L. (2001). Word length effect in speech production of Chinese. Acta Psychologica Sinica, 33, 214−218.
[庄捷, 周晓林. (2001). 言语产生中的词长效应. 心理学报, 33, 214−218.]
```

### 3.5 图书格式

```
著者. (出版年). 书名（斜体）. 出版社.
```

**示例：**
```
张三. (2008). 警惕心理学中的生物化倾向. 北京：未名出版社.
```

### 3.6 编著中的章节格式

```
章节著者. (年). 章节标题. 见 编者 (编), 书名（斜体）(pp. 页码). 出版社.
```

- 英文编者：名前姓后；中文编者：姓前名后（无需颠倒）。
- 页码范围前单页用 p.，多页用 pp.。

**示例：**
```
张三. (2008). 汉代心理学思想. 见 李四 (主编), 心理学通史：第2卷. 中国古代心理学 (pp. 12−34). 新华出版社.
```

### 3.7 学位论文格式

```
著者. (年). 论文题目（斜体）（博士/硕士学位论文）. 学位授予单位[, 城市（若单位名含城市可省略）].
```

- 英文论文需加 Unpublished，中文不加。
- 硕士论文：master's thesis；博士论文：doctoral dissertation。

**示例：**
```
Yu, L. (2000). Phonological representation and processing in Chinese spoken language production (Unpublished doctoral dissertation). Beijing Normal University.
[余林. (2000). 汉语语言产生中的语音表征与加工 (博士学位论文). 北京师范大学.]
```

### 3.8 报纸格式

```
著者. (年-月-日). 文题. 报纸名, p./pp. 页码.
```

**示例：**
```
张三, 李四. (2008-08-08). 中国心理学与奥林匹克. 新华日报, p2, 5−7.
```

### 3.9 网络资源格式

提供 DOI（优先）或 URL，格式为超链接形式，不再使用 "DOI:" 标签。

### 3.10 常用缩写

| 缩写 | 含义 |
|------|------|
| chap. | 章 |
| ed. | 版 |
| Rev. ed. | 修订版 |
| 2nd ed. | 第2版 |
| Ed./Eds. | 编者 |
| Trans. | 译者 |
| n.d. | 无日期 |
| p./pp. | 页/页码范围 |
| Vol./vols. | 卷 |
| No. | 第 |
| Pt. | 部分 |
| Suppl. | 增刊 |

---

## 四、元分析中的文献

元分析中用到的研究报告，在文献前加星号 *，并在文献列表开头注明 * 表示元分析用到的文献。
